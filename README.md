## 使用说明
1. 进入任意一个B站直播间
2. 按`F12`,选择`network`或`网络`
3. 发送一条弹幕
4. 在`network`或`网络`选项卡右键,点击`将所有内容另存为har`
5. 将下载的har文件放到程序的同一目录下,保存为`bilibili.har`
6. 启动程序!!!

弹幕默认是发送到`呵呵`的直播间,要改直播间的话,在程序的文本框中输入房间号,再点击`Misaka Ⅱ`.
